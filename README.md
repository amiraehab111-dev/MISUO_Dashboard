# MISUO Sales Performance Analysis & Dashboard

An end-to-end data analysis project for **MISUO** sales and operations. This project transforms raw, unorganized sales data into actionable business insights using Excel.

## 📁 Project Structure & Workflow
The workbook is organized into specialized sheets representing the full data pipeline:
1. **البيانات (Raw Data):** The original, uncleaned customer orders and sales records.
2. **الارشادات (Guidelines):** Reference guide and business rules for data processing.
3. **cleand_data (Data Cleaning):** The processed dataset after handling missing values, standardizing dates, and correcting formats.
4. **MISUO_pivot (Data Aggregation):** Pivot tables summarizing sales by product, platform (like Homzmart), regions, and delivery status.
5. **MISUO_Dashboard (Data Visualization):** An interactive dynamic dashboard designed for stakeholders to monitor key performance indicators (KPIs), fulfillment rates, and revenue trends.

## 🔍 Key Insights Covered
- Sales distribution across different e-commerce platforms.
- Order fulfillment status (Delivered vs. Failed deliveries).
- Product performance tracking (Beds, Tables, etc.) with specific attributes (dimensions, colors, materials).
- Delivery partner and logistics performance analysis.
